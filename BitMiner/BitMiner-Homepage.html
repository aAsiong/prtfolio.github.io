Welcome to

BitMiner is a web application created to track the status and conditon of devices used to mine Bitcoin or any tasks that can be run without any human operator.

How It Works

BitMiner main functionality is to offer a dashboard for it's users to insert their devices for ease of management and temperature tracking. This can be done with the user downloading the WeMos code provided by the developers and then running it in their devices. The inserted code will connect to the database to which data will be stored and later be displayed in the user's dashboard
